# 🌐 TemCode — 모던 웹 서비스 플랫폼

> Toss와 Apple의 감성을 결합한 미니멀하고 신뢰감 있는 웹 서비스

---

## 🧭 개요
**TemCode**는 깔끔한 디자인과 부드러운 인터랙션을 중심으로 한 웹 서비스 플랫폼입니다.  
모던한 감각과 안정적인 사용자 경험(UX)을 기반으로,  
기업 웹사이트나 개인 프로젝트, 포트폴리오 등에 손쉽게 적용할 수 있습니다.

---

## ✨ 주요 기능
- **미니멀한 디자인** — Toss · Apple 스타일 기반의 심플한 레이아웃  
- **반응형 웹** — 모든 기기(모바일, 태블릿, PC)에 최적화  
- **다중 페이지 구성** —  
  - 서비스 소개  
  - FAQ  
  - 고객센터 / 문의  
  - 로그인 및 회원가입  
  - 템플릿 갤러리  
  - 후기(테스트모니얼)  
- **매끄러운 사용자 경험** — 자연스러운 전환 애니메이션과 직관적 UI  

---

## 🛠️ 기술 스택
- **HTML5 / CSS3 / JavaScript**
- 반응형 레이아웃 및 UI 애니메이션
- 접근성과 심미성을 고려한 구조 설계

---

## 💡 프로젝트 목표
TemCode는 단순한 웹사이트 템플릿을 넘어,  
**신뢰감·디자인·사용성**을 모두 갖춘 모던 웹 플랫폼을 지향합니다.
